Supported Formulas

Use these inside column JSON source expressions in the column editor dialog.
4 formulas
CURRENT_TIMESTAMP() 0 args
Returns the current date & time formatted as yyyy-MM-dd HH:mm:ss.
Example 
CURRENT_TIMESTAMP()
FIND(csv_column, json_path, json_element) 3 args
Scans the JSON at json_path (object or array), finds the first entry where the field at that path equals the value of csv_column, and returns json_element from the same object. Returns null if no match.
csv_column json_path json_element
Example 
FIND(appid, results.app_info.appid, app_name)
The compare uses the value of the CSV column. If the JSON node is an array, it checks each element.
CONCAT(arg1, arg2, …) N args
Concatenates arguments. Each argument can be a CSV column name (its value is used) or a single-quoted string literal. Missing or null column values are treated as empty strings.
columnName 'literal'
Example 
CONCAT(appid, ' | ', app_name)
Use single quotes for literals: ' - '. To include a single quote inside a literal, escape it as \'.
JSON(json_path) 1 arg
Returns the JSON at the given path as a string (object/array/value), or null if the path doesn’t exist.
json_path
Example 
JSON(results.app_info)
Literals must be wrapped in single quotes. Column names reference the current row context.
